# Autonomous Robot Navigation System

The objective of this project is to implement autonomous navigation systems with limited hardware resources while still maintaining real-time decision making and accuracy. The code runs on a GoPiGo3 robot with a Model 4B Raspberry Pi with 4GB RAM for processing instructions. 

## Background
Autonomous navigation is an increasingly popular topic in computer vision research, as it is the technology behind self-driving vehicles. The National Highway Traffic Safety Administration (NHTSA) defines 6 levels of autonomous driving, with Level 0 being mostly driver-operated, and Level 5 being entirely autonomous. Current research in this field uses open vocabulary perception and LiDAR as a baseline to focus on advancements in other technologies towards Level 3-5 autonomous driving.
